# P&T 공정 SOP / 지식베이스 (데모용 가상 문서)

> 본 문서는 포트폴리오 데모용으로 작성된 가상의 제조 공정 지식 문서입니다. 실제 SK하이닉스 자료가 아닙니다.

## 1. HBM TSV 적층 본딩 공정 (PKG-HBM-01)
HBM은 여러 장의 DRAM die를 TSV(Through-Silicon Via)로 수직 적층하여 제조한다.
적층 본딩 시 핵심 관리 항목은 본딩 온도, 가압력, 정렬(alignment) 정밀도이다.
- 권장 본딩 온도: 230~260°C
- 가압력 관리 범위: 30~50 N
- 정렬 오차 허용치: ±3 µm 이내
정렬 오차가 허용치를 초과하면 TSV 접합 불량(open/short)이 발생하며, 이는 최종 테스트에서 채널 불량으로 검출된다.

## 2. 와이어 본딩 불량 대응 (PKG-WB-02)
와이어 본딩 공정에서 빈번한 불량 유형은 다음과 같다.
- Non-stick on pad (NSOP): 패드 오염 또는 본딩 온도 저하가 주원인. 본딩 온도와 패드 청정도를 우선 점검한다.
- Wire sweep: 몰딩 공정 수지 흐름 과다가 원인. 몰딩 압력과 점도를 점검한다.
NSOP 발생률이 0.5%를 초과하면 즉시 본딩 설비의 EFO(전기방전) 전류와 캐필러리 상태를 점검해야 한다.

## 3. 최종 테스트(Final Test) 절차 (TEST-FT-03)
최종 테스트는 패키징 완료된 칩에 대해 전기적 특성과 기능을 검사한다.
- 테스트 단계: Contact Test → DC Parametric → Function Test → Speed Binning
- Contact Test 실패가 급증하면 핸들러 소켓 접촉 불량 또는 보드 오염을 우선 의심한다.
- Function Test의 채널 불량이 특정 die 위치에 집중되면 적층 본딩(PKG-HBM-01) 정렬 문제를 역추적한다.

## 4. FDC 알람 대응 가이드 (FDC-GUIDE-04)
FDC(Fault Detection & Classification)는 설비 센서 데이터로 이상을 탐지·분류한다.
알람 심각도는 Critical / Major / Minor 3단계로 구분한다.
- Critical: 즉시 설비 정지(interlock) 후 엔지니어 호출. 로트 홀드 처리.
- Major: 진행 중 로트 완료 후 설비 점검. 동일 알람 3회 반복 시 Critical로 격상.
- Minor: 모니터링 강화. 추세(trend) 악화 시 Major로 격상.
온도 센서 알람은 대부분 챔버 히터 또는 냉각수 유량 문제와 연관된다.

## 5. SPC 관리도 이탈 대응 (SPC-05)
SPC(Statistical Process Control)는 공정 계측치를 관리도로 모니터링한다.
- 관리한계(UCL/LCL) 이탈: 1점이라도 벗어나면 이상원인(assignable cause) 조사.
- 연속 7점이 중심선 한쪽: 공정 평균 이동(shift) 의심. 설비 셋업/자재 lot 변경 이력 확인.
- 추세(연속 상승/하락): 설비 마모 또는 드리프트 의심.
SPC 이탈 발생 시 MES에서 해당 로트의 설비 이력과 자재 lot을 함께 조회하여 원인을 좁힌다.

## 6. 수율(Yield) 저하 원인분석 표준 절차 (YIELD-06)
수율 저하 발생 시 표준 원인분석 순서는 다음과 같다.
1. 불량 유형 분류(bin 분석)로 어떤 테스트 항목에서 떨어졌는지 확인
2. 불량의 공간적 분포(wafer/lot map) 확인 — 특정 위치 집중 여부
3. 해당 시점의 FDC 알람·SPC 이탈 이력 교차 확인
4. 설비/자재 lot 변경 이력 확인
5. 유사 과거 사례 검색 후 조치 적용
채널 불량이 특정 die 위치에 집중되면 TSV 적층 정렬(PKG-HBM-01)을 최우선 의심한다.
